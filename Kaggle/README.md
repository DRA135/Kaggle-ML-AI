이 폴더는 Kaggle 대회와 관련된 실험 코드, 모델링 과정, 분석 결과를 정리한 공간입니다.  

## 1) Jane Street Real-Time Market Data Forecasting
실시간 증권 시장 데이터를 활용해 매수/매도 의사결정 신호를 예측하는 시계열 기반 모델링 프로젝트입니다.  
주요 내용:
- Feature engineering 및 시계열 전처리  
- MLP, LSTM, Transformer 기반 예측  
- 실험 로그 및 성능 비교  
- 리스크 관리 관점의 모델 평가

## 2) CIBMTR Survival Analysis
조혈모세포 이식 환자의 생존 확률을 예측하는 생존분석 기반 프로젝트입니다.  
주요 내용:
- Kaplan–Meier, Cox 모델 활용  
- 임상 변수 기반 Survival Curve 분석  
- 위험요인 변수 중요도 해석  
- 모델별 성능 비교 및 시각화
